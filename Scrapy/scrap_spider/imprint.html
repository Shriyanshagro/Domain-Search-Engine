System Center 2016 Configuration Manager Reporting Unleashed

The heart of Microsoft System Center Configuration Manager is its SQL Server database. This database contains a wealth of information about users, computers, hardware and software inventory, installed applications, compliance, and more. Microsoft has provided excellent tools for extracting this information in meaningful ways, but it hasn't given System Center users the knowledge they need to use these tools. System Center 2016 Configuration Manager Reporting Unleashed fills that gap, guiding administrators through every step and technique they need to get crucial information for solving problems and optimizing their IT infrastructure.
