HDT Vocabulary

The HDT vocabulary is a set of properties in order to provide a Header which describes the data of an RDF HDT data set.

Last update:

30 March 2011

Namespace URI:

http://purl.org/HDT/hdt#

Abstract

The HDT vocabulary is a set of properties in order to provide a Header which describes the data of an RDF HDT data set.
