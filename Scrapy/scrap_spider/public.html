Advisory: HP ProLiant Gen9 Servers - After Executing the VMware Power Pic Smart Component Firmware Flash, the Server May Stop Responding

SUPPORT COMMUNICATION - CUSTOMER ADVISORY

Document ID: c04476803

Version: 1

Advisory: HP ProLiant Gen9 Servers - After Executing the VMware Power Pic Smart Component Firmware Flash, the Server May Stop Responding
NOTICE: The information in this document, including products and software versions, is current as of the Release Date. This document is subject to change without notice.

Release Date: 2014-10-11

Last Updated: 2014-10-11

DESCRIPTION

After executing the VMware Power Pic Smart Component firmware flash through any version of HP Smart Update Manager (HP SUM) or on the Integrated Lights Out 4 (iLO 4) command line, a ProLiant Gen9 server may become unresponsive if the server is power-cycled during the flashing process for the Power Management Controller.

SCOPE

Any HP ProLiant Gen9-series server with Integrated Lights Out 4 (iLO 4), and running VMware ESXi 5.0, VMware vSphere 5.1, or VMware VSphere 5.5.

RESOLUTION

A future version of VMware Power Management Controller Firmware for HP ProLiant Gen9 servers will correct this issue.

As a temporary workaround, extract the firmware binary from the Smart Component and flash the firmware to update via the Integrated Lights Out 4 (iLO 4) interface:

  1. Copy the .scexe to a Linux server or a VMware server.
  2. Run the following command: *.scexe - - unpack=<target directory>.
  3. Use the Signed/*s.hex payload file to flash through iLO 4.

This advisory will be updated when additional information becomes available.



RECEIVE PROACTIVE UPDATES : Receive support alerts (such as Customer Advisories), as well as updates on drivers, software, firmware, and customer replaceable components, proactively via e-mail through HP Subscriber's Choice. Sign up for Subscriber's Choice at the following URL: http://www.hp.com/go/myadvisory

SEARCH TIP : For hints on locating similar documents on HP.com, refer to the Search Tips document .

Hardware Platforms Affected: HP ProLiant DL360 Gen9 Server
Operating Systems Affected: VMware ESXi 5.0(Standard HP Product), VMware ESXi 5.1(Standard HP Product), VMware ESXi 5.5
