ライセンス体系

  • SafeGuard Enterprise Encryption

    SafeGuard Enterprise Encryption バンドルには、SafeGuard Disk Encryption と SafeGuard File Encryption が単一のライセンスに含まれています。これは暗号化を初めて使用するお客様で、さまざまな OS を使用し、データがどこにあっても確実に暗号化されることを必要とするお客様に最適のソリューションです。

  • SafeGuard Disk Encryption

    SafeGuard Disk Encryption は、SafeGuard Native Device Encryption for BitLocker/FileVault 2 の管理と、ソフォスのフルディスク暗号化機能である SafeGuard Device Encryption を組み合わせています。これは、OS の種類に関係なく社内のすべてのノートブック PC およびデスクトップ PC にフルディスク暗号化を適用する必要があるお客様に最適なソリューションです。本ライセンスは、ユーザーベースのサブスクリプションライセンスです。

  • SafeGuard File Encryption

    SafeGuard File Encryption は、リムーバブルメディア、クラウドストレージ、およびネットワーク共有にあるデータのファイル暗号化を提供します。これは、エンドユーザーのデバイスから転送されるデータが確実に暗号化されることを必要とするお客様に最適です。
