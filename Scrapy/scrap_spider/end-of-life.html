Virtual Intranet Access (VIA)
Aruba always supports the two most recent VIA major/minor releases for a given operating system. For example, if VIA 2.1.1 is the current release for Windows, then VIA 2.1.x and VIA 2.0.x will be supported. When VIA 2.2 is then released, VIA 2.0.x would move to immediate end-of-support status. End of support for VIA implies that Aruba will no longer issue maintain or issue updates for that release.
