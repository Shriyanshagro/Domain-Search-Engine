    Mail Server Security with Leading Malware and Spam Protection

    More than 90% of targeted attacks or advanced persistent threats (APTs) begin with a spear phishing email—a type of phishing attack that targets specific groups or individuals. Unlike other leading solutions, ScanMail blocks targeted attacks, as well as spam, phishing, and malware with global threat intelligence from the Trend Micro™ Smart Protection Network™. And with optional data loss prevention, it can discover sensitive data in the mail store and simplify compliance initiatives.

    Protection against Targeted Attacks

    New targeted attack protections are built into to ScanMail and enhanced with Deep Discovery Advisor. Web reputation, document exploit detection, sandbox execution analysis, and custom local threat intelligence allow you to create a network defense against spear phishing attacks associated with APTs and other targeted threats.

    • Detects known and unknown exploits in PDF, MS Office and other documents
    • Performs malware execution analysis, provides custom threat intelligence and delivers adaptive security updates via optional Deep Discovery Advisor integration
    • Stop threats from entering your environment with immediate protection based on global threat intelligence from Trend Micro™ Smart Protection Network cloud-based security

    See more on custom APT security

    Unique Reputation Technology to Block Spam, Phishing and Malware

    Powered by the Trend Micro™ Smart Protection Network™, ScanMail Suite stops spam, phishing, and targeted attacks based on real-time reputation ratings.

    • Delivers immediate protection from emerging threats by using big data analytics and predictive technology to correlate file, web, and email reputation data
    • Checks for malicious links within both the email body and attachments to block phishing attacks via enhanced web reputation
    • Drops up to 85 percent of all incoming email using email sender reputation to free network resources
    • Integrates with Microsoft Outlook junk email folder
    • Stops more spam than other email server security solutions according to independent tests

    • Accelerates setup and improves accuracy with 100+ compliance templates
    • Enables granular Active Directory-based policy enforcement
    • Enables compliance personnel to centrally manage DLP policies and violations across other Trend Micro products from endpoint to gateway with Trend Micro Control Manager™

    You Need a Network Defense to Thwart a Targeted Attack

    Trend Micro messaging and email server security products provide protection against targeted attacks with enhanced web reputation, a new detection engine, and new threat analysis appliance to block highly targeted email attacks by using exploit detection and sandboxing. As a component of the Trend Micro™ Network Defense solution, the integration of these components provides a network defense that enables you to detect, analyze, adapt and respond to targeted attacks. The following defenses are found in:

    • InterScan Messaging Security
    • ScanMail for Microsoft Exchange  
    • ScanMail Suite for IBM Domino
    • Enterprise Security for Endpoints and Mail Servers
    • Enterprise Security for Communication and Collaboration
    • Enterprise Security Suite

    Enhanced Web Reputation

    • Web reputation is powered by the big data analytics and predictive technology of Smart Protection Network global threat intelligence
    • Part of Smart Protection Network, SmartScan technology scans URLs embedded in emails and attachments and blocks URLs leading to malicious sites

    Advanced Threat Scan Engine

    • Detects advanced malware in PDFs, MS Office, and other documents using static and heuristic logic to detect and examine abnormalities in addition to known and zero-day exploits
    • Scans the Exchange mail store for targeted threats that may have passed through before protection was available
    • Quarantines suspicious attachments for automatic sandbox execution analysis to occur in-line without impacting the delivery of majority of messages (requires integration with optional Trend Micro™ Deep Discovery Advisor)

    Deep Discovery Advisor integrates with ScanMail as part of the Trend Micro™ Network Defense Solution

    • Provides automatic in-depth simulation and analysis of potentially malicious attachments including executables and common office documents in a secure sandbox environment
    • Allows customers to create and analyze multiple fully custom target images that precisely match their host environments
    • Analyzes logs of Trend Micro products and third-party solutions combined with Trend Micro threat intelligence—so you gain in-depth insights that support risk-based incident assessment, containment and remediation
    • Issues custom security updates for adaptive protection and remediation by ScanMail, Trend Micro products, and third-party security layers
