Windows Authentication & WCF service calls

up vote 0 down vote favorite

I have the following code setting up my serivce:

        BasicHttpBinding binding = new BasicHttpBinding(BasicHttpSecurityMode.TransportCredentialOnly);
        binding.Security.Transport.ClientCredentialType = HttpClientCredentialType.Windows;
        binding.SendTimeout = TimeSpan.FromMinutes(3.0);

        SearchManagerClient client = new SearchManagerClient(binding, new EndpointAddress("http://aee/wcf/SearchManager.svc"));

When I run the methods to get the data I want I end up with 0 results (on the dev machine and on remote machine)

I then go and add:

            WindowsIdentity id = (WindowsIdentity)HttpContext.User.Identity;
            using (WindowsImpersonationContext wic = id.Impersonate())
            {
                   [All the inner code with the same code as above with the same methods]
            }

When the Impersonate option is called on the local dev machine it returns 19 results (which is the expected) but on a remote machine it returns

[WebException: The remote server returned an error: (401) Unauthorized.] System.Net.HttpWebRequest.GetResponse() +1740
System.ServiceModel.Channels.HttpChannelRequest.WaitForReply(TimeSpan timeout) +75

[MessageSecurityException: The HTTP request is unauthorized with client authentication scheme 'Negotiate'. The authentication header received from the server was 'Negotiate,NTLM'.]

This error is preventing me from moving forward which really is annoying.

The website uses windows authentication and I am needing it so whatever user uses the site the WCF service uses the same client so as to only get the information that the client has access to. However I spent all last week trying to get it to work but nothing seems to be able to. Any help would be very much appreciated! If you need any more information please let me know, I do not work with services all that often and it could be possible I forgot to include some information.

c# asp.net wcf windows-authentication basichttpbinding
