NAME

Audio::Metadata::Flac - FLAC format support for Audio::Metadata

§

DESCRIPTION

This module implements FLAC format support for Audio::Metadata. It's implemented in pure Perl, doesn't rely on external libraries and allows metadata of arbitary length, rewriting the file, if necessary. For description and usage see Audio::Metadata.
