Scalar-List-Utils-1.45

Changes for version 1.45

  • CHANGES
    • Renamed existing uniq() to uniqstr()
    • Canonicalise undef to {empty string,zero} in uniq{str,num}()
    • Add a new uniq() with more DWIMish semantics around undef
  • BUGFIXES
    • Fix uses of GET magic by the uniq*() family of functions. GET magic is now always invoked exactly once if it exists.
