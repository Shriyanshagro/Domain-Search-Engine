Key Features

Continuous Measurement and TrackingDetailed AnalyticsSingle User Experience ScoreUser FlowSimple ImplementationSelf-service SaaS

Continuous Measurement and Tracking

Continually measure the user experience from the user's perspective. Measure and monitor everything the user does and everywhere the user goes within the app and everything that happens along the way. Identify issues having the most impact on the user experience and improve what matters most.

Detailed Analytics

Leverage mobile analytics with actionable insights so you can prioritize development efforts that improve the user experience. Know how many users were impacted by which user action, OS, device and app version.

Single User Experience Score

Gain visibility into the health of your mobile app with a single user experience score. FunDex gives you a top-level view of launch times, crashes, errors and resource usage that negatively impact the user experience with ways to explore the root cause of each issue.

User Flow

Follow the funnel of user interactions with your mobile app, screen by screen. Drill down to data about user actions to understand when and why users exited or abandoned the mobile app so you can optimize to keep users fully engaged with the app.

Simple Implementation

Onboard quickly and keep your release cycles on track! Implementation is simple: you don't need to change code or add any tags.

Self-service SaaS

Offer your team the self-service SaaS interface that they are used to. Increase mobile app monitoring efficiency and satisfaction, and avoid the cost of hardware and software installation and maintenance.
