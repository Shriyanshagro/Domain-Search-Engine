    For anyone looking for a workaround:

    1. Make sure the AAD Connect wizard is not running.

    2. Open this file: %ProgramData%\AADConnect\PersistedState.xml

    3. Find a section that looks like this:

    <PersistedStateElement>
          <Key>IAdfsContext.TargetForestDomainName</Key>
          <Value>contoso.com</Value>
    </PersistedStateElement>

    Change the <Value> element to look like this:

    <PersistedStateElement>
          <Key>IAdfsContext.TargetForestDomainName</Key>
          <Value>contoso.com,contoso.com</Value>
     </PersistedStateElement>

    Save the file and restart the AAD Connect wizard.
