About dependency-properties

A property in WPF and Silverlight that supports databinding, including styles, inheritance, animation, and default values. It allows you to set attached properties on DependencyObject.

Dependency property: A property that is backed by a DependencyProperty.

The purpose of dependency properties is to provide a way to compute the value of a property based on the value of other inputs.

Ref: Dependency Property Overview
