    Security built for the cloud, delivered from the cloud

    In the cloud, security is a shared responsibility. Cloud providers are responsible for the security of the physical and network infrastructure to the hypervisor layer. And you must protect what you put in the cloud: operating systems, applications, and data.

    Trend Micro Deep Security augments cloud provider security controls and certifications. Tight integration with leading cloud providers results in fast deployment and the elimination of security gaps. This lets you protect workloads without hampering agility or flexibility.

    Deep Security as a Service provides all the protection of Deep Security - without all the work. We manage product and kernel updates, set up and maintain the security database, and administer the Deep Security manager for you – letting you focus on your core business.

    Overview of Deep Security as a Service


    We’ve worked closely with AWS, Microsoft Azure, VMware vCloud Air, and others, to ensure that Deep Security can seamlessly protect instances, with a light-weight agent, even when auto-scaling. Deep Security as a Service gives you:

    • Less work. Trend Micro manages the Deep Security manager, product updates, and the security database for you so you can focus on your business.
    • Secure DevOps. Deployment scripts for configuration management tools like Chef, Puppet, and OpsWorks are provided so you can build security into your operations.
    • Host-based controls. Deep Security’s host-based approach means you can customize protection on workloads based on tags and automatically protect as new instances come online.
    • Proactive protection. IPS capabilities enable you to prevent attacks, not just be alerted when intrusions have been detected.
    • Multiple controls, single light-weight agent. Reduce management and procurement headaches with six robust security controls in a single product. The agent is workload-aware and with tags you can customize the controls and rules applied, minimizing agent footprint.

    Agile, proactive cloud workload security

    Not only does Trend Micro Deep Security as a Service provide the most complete set of host-based security capabilities available for the cloud, seamless integration with cloud providers ensures security won’t get in your way.

    Thousands of customers choose Trend Micro Deep Security to protect millions of servers globally. Host-based protection includes:

    • Intrusion Detection and Prevention: Inspect traffic to detect and block suspicious activity and prevent exploitation of vulnerabilities.
    • Anti-Malware: Protect your workloads against malicious software.
    • Bidirectional Firewall: Complement security groups with fine-tuned control of ingress and egress points and maintain firewall logs for audits.
    • Web Reputation: Block access to malicious domains and known communication & control servers (C&C) used by criminals.
    • Integrity Monitoring: Track all changes made to an instance and receive alerts about suspicious or malicious changes.
    • Log Inspection: Capture and analyze system logs to provide audit evidence for PCI-DSS or internal requirements.
    • Events and Reports: Quickly provide audit evidence with Deep Security’s 16 different types of reports including user reports, security module specific reports, and general reports with custom filtering.
