Finally, the problem was the storyboard.

1.- The error i reproduced with this procedure on device: Technical Q&A QA1764 with this the app crashes immediately at launch, and a error message printed to the console (look in organizer)

2.- When you localized the storyboard with one language (spanish in my case), main .storyboard file was missing in archive. It's seems this is a error of xcode.
