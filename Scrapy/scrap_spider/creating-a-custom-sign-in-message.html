Creating a custom sign in message

  • mac
  • windows
  • linux
  • all

You can create a custom message that users will see on the sign in page.

  1. As a site admin, sign in to your GitHub Enterprise instance at http(s)://[hostname]/login.

  2. In the upper-right corner of any page, click .

  3. In the left sidebar, click Admin Center.

  4. In the left sidebar, click Messages.

  5. In the Sign in message field, type the message you'd like users to see. Markdown is supported.

  6. Click the Preview button under the Sign in message field to see the rendered message.

  7. Review the rendered message.

  8. At the bottom of the page, click Save changes.
