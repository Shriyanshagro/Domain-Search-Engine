Red Hat Cloud Suite highlights

What is Red Hat Cloud Suite?

Red Hat Cloud Suite provides a container-based application development platform, which is built on a massively-scalable cloud infrastructure and managed through a common management solution. Red Hat Cloud Suite delivers public cloud-like services to developers and to businesses while maintaining control and visibility.

With Red Hat Cloud Suite, users can build a private cloud based on Red Hat OpenStack Platform with public cloud-like scalability, or on Red Hat Enterprise Virtualization with high-performance virtualization. Both of these choices provide secure and scalable foundations for hosting the development platform known as OpenShift Enterprise, which automates the development and administration of container-based applications.

Red Hat Cloud Suite Features

Modernize apps and run them where you need them

Develop modern applications on a container-based application platform. Run workloads in containers or virtual machines on a variety of different platforms.

All the parts you need in one solution

We've tightly integrated cloud technologies so you don't have to piece together a solution. Fully-supported components work together seamlessly to help you build cloud environments, write apps, and maintain integrated management, governance, and workload portability.

Manage your IT environment from a single place

Use the same management tool across virtualization, private cloud, app-development platform, containers, and public cloud.

Update your infrastructure

Build a complete and easy-to-manage OpenStack Infrastructure-as-a-Service (IaaS) private cloud on top of what you already have, allowing you to deliver public cloud scalability and speed up service delivery.

The QuickStart Cloud Installer is a web-based graphical user interface for provisioning cloud products. The QuickStart Cloud Installer provides a streamlined prescriptive deployment for the following products:

For a basic installation, you'll need:

External Network Connection

To subscribe and enable repositories hosted by Red Hat.

System Requirements

The system and environment must meet the prerequisites.

Four Hours

Installation will take about 4 hours.

instructions

  1. 1

    Install the QuickStart Cloud Installer

    • Download the Quickstart Cloud Installer (QuickStart Cloud Installer BETA ISO).
    • Install and select Install Red Hat QuickStart Cloud Installer 1.0
    • Login as the root user
    • Configure the networking details for the QuickStart Cloud Installer machine
    • Change the default password for the administration user
  2. 2

    Select between Red Hat Enterprise Virtualization and Red Hat OpenStack Platform for your private cloud infrastructure

    • For some information on when you would use one versus the other, please see the Red Hat Cloud Suite Planning and Engineering Guide
  3. 3

    For deployment using Red Hat Enterprise Virtualization :

    • Create a node to be used as the Red Hat Enterprise Virtualization Manager and one for the hypervisor host.
    • Verify the discovered nodes.
    • Set up shared NSF storage.
    • In the QuickStart Cloud Installer menu, choose New Deployment and
      • select Red Hat Enterprise Virtualization.
      • select Red Hat CloudForms, should you elect to use it.
    • Configure the Satellite and Red Hat Enterprise Virtualization components and then the CloudForms and subscriptions component.
    • Verify the installation configuration settings and click on Deploy.
    • Verify the deployment.
  4. 4

    For deployment using Red Hat OpenStack Platform :

    • Download and install Red Hat Openstack Platform Director (Red Hat OpenStack Platform Undercloud Installer ISO).
    • In the QuickStart Cloud Installer menu, choose New Deployment and:
      • select Red Hat OpenStack Platform.
      • select Red Hat CloudForms, should you elect to use it.
    • Configure the Satellite and Red Hat OpenStack Platform components and then the CloudForms and subscriptions component.
    • Verify the installation configuration settings and click on Deploy.
    • Verify the deployment.
