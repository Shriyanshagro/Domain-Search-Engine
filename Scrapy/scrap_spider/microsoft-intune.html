How to buy Microsoft Intune

Microsoft Intune subscriptions are licensed on a per-user basis so that you don’t have to predict the number of devices per user. Intune is also a component of Microsoft’s Enterprise Mobility Suite (EMS), the most cost-effective way to acquire all of the included cloud services: Azure Active Directory Premium, Azure Rights Management, and Intune.
