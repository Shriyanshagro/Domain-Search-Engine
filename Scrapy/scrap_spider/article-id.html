La coppia perfetta 2: Windows Server 2012 R2 Essentials con HP ProLiant Gen9

da MassimilianoG in data ‎09-09-2015 03:22 PM

Come per tutte le aziende, anche per i vostri clienti PMI è necessario occuparsi di più dell’attività principale che della gestione dell’Information Technology (IT). La tecnologia deve aiutare l’azienda a essere più produttiva, accedere più facilmente alle applicazioni dappertutto e tramite ogni dispositivo, proteggere i dati e fornire un percorso di preferenza per l’integrazione col cloud. Una soluzione server Microsoft-HP può fare tutto questo e anche di più! Vediamo di seguito le caratteristiche di Windows Server 2012 R2 in versione Essentials e di 3 server HP ProLiant Gen9 ideali da abbinare con questo sistema operativo.

 

Windows Server 2012 R2 Essentials, l’ultima versione di Windows Small Business Server Essentials, offre agli utenti una soluzione server flessibile, a basso costo e facile da usare per le PMI che hanno fino a 25 utenti e fino a 50 dispositivi. Questa edizione di Windows Server 2012 R2 è progettata per aiutare le PMI ad avere standard tecnologici simili a quelli delle grazie aziende ma senzi grandi costi e senza le stesse complessità di gestione. L’edizione Essentials fornisce anche molta flessibilità di utilizzo e possibilità di scelta, che permettono alle piccole aziende di poter operare sia in ambiente cloud che tradizionale. Le tre caratteristiche principali di Windows Server 2012 R2 Essentials, in grado di rispondere alle sempre più pressanti esigenze dei clienti PMI, sono: 1) protezione dei dati, 2) accesso remoto sicuro e 3) integrazione coi servizi cloud.

 

Ecco in questa foto quelli che, secondo noi, possiamo definire i server perfetti per le PMI da abbinare a Windows Server 2012 R2 Essentials.

 

 

Di HP ProLiant DL120 Gen9 ne avevamo già parlato nel precedente blog della serie “La coppia perfetta”, quando lo avevamo consigliato insieme a Windows Server 2012 R2 Foundation. Quindi, voi direte “c’è un errore”. Nessun errore; questo server si abbina perfettamente con tutte e due le edizioni. Non è soltanto un’ottima soluzione per le PMI agli inizi della loro attività e che utilizzano Foundation, ma è anche perfetta per le aziende che vogliono crescere e che utilizzano Essentials. HP ProLiant DL120 Gen9 è progettazione di classe enterprise in un formato ad alta densità socket 1U/1 e offre una perfetta combinazione di ridondanza, prestazioni ed espandibilità. Questo server supporta un processore Intel® Xeon® E5-2600 v3.

 

Se i vostri clienti PMI hanno bisogno di un server con supporto fino a due processori, HP ProLiant DL160 Gen9,con il supporto del processore Intel® Xeon® E5-2626 v3, è l’opzione ideale. Inoltre supporta diverse opzioni di FlexibleLOM, fino al 92% di efficienza energetica a 900W e configurazioni Energy Star a supporto dell’SKU. Le PMI possono ottenere tutto ciò che desiderano da DL160 Gen9, una piattaforma integrata progettata per scalabilità ed efficienza. —a fully integrated platform designed to perform and scale. Con HP Integrated Lights-Out (iLO) fornito di serie per la gestione dell’infrastruttura IT, 16 slot di memoria DDR4, 8 drive e due porte NIC, il server ProLiant DL160 Gen9 offre il set di caratteristiche ideale per le esigenze aziendali di tutte le PMI.

 

Infine, per i vostri clienti PMI alla ricerca di un server tower da abbinare a Windows Server 2012 R2 Essentials, consigliamo HP ProLiant ML110 Gen9, il server tower 4.5U, dotato di un singolo processore, che include fino a cinque slot PCIe per l’espansione, fino a otto dischi fissi Large Form Factor (LFF) o sedici dischi Small Form Factor (SFF). Inoltre, è dotato di otto slot DIMM DDR4 con supporto HP DDR4 SmartMemory con una capacità massima di 256 GB, rendendolo il server ideale per le piccole e medie aziende e le filiali.

 

 

**NOTA: Se i vostri clienti PMI preferiscono invece Windows Server 2012 R2 Foundation vi invitiamo a leggere il post La coppia perfetta: Windows Server 2012 R2 Foundation con ProLiant Gen9”.
