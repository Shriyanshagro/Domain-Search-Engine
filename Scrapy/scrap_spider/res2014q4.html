Storage Notes

  • All storage accessed with 4 x 8 Gb Fibre Channel links. And the SUT was accessed via four fibre channel controllers
  • 1 x Huawei Dorado5100 was used with 4 disk enclousures (96 drives used) attached to the Dorado5100 via 4 x 8 Gb drive ports. A RAID10 array containing 24 x 200 GB SSD drives is configured on each Dorado5100 disk enclosure. There are 4 RAID10 arrays on Dorado5100. Each RAID10 array has only 4 LUNs,each LUN is used to store 1 tile of VM images, mailserver datastore and infraserver datastore. There are 16 tiles on this Dorado5100 (16 LUNs used).
  • 3 x HUASY Dorado2100 were used via 3 x 8 Gb drive port each. Each HUASY Dorado2100 contains 24 x 100 GB SSD drives. There are 3 RAID10 arrays, each consisting of 24 drives (72 drives used). One RAID10 array has 6 LUNs for 6 dbserver's datastore,the second RAID10 array contains 4 LUNs for 4 tiles of VM images, mailserver datastore and infraserver datastore, the last RAID10 array contains 3 LUNs for 3 tiles of VM images, mailserver datastore and infraserver datastore.
  • The Server RH5885H V3 of SUT contains 4 x 300 GB SAS drives. A RAID5 array consists of 4 hard disks. This array is used as local disk for installing Virtualization Platform. And the boot device is on the local disk /dev/sda.
