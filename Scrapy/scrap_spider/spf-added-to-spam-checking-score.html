SPF added to spam checking score

Rob Mueller – 30 March 2006

Spam checking for full/enhanced accounts via SpamAssassin has been updated to include SPF checks as an additional score.

http://www.openspf.org/

We're currently using the default SpamAssassin scores for these checks:

score SPF_HELO_SOFTFAIL 2.432
score SPF_NEUTRAL 1.069
score SPF_SOFTFAIL 1.384
