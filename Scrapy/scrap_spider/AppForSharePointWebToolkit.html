App for SharePoint Web Toolkit (for...

3.1.2

This package adds SharePoint client object model assembly references and token helper code to the web project for SharePoint 2013. These items make it easier to develop the web project if the web project needs to access data from the app for SharePoint 2013.

To install App for SharePoint Web Toolkit (for SharePoint 2013), run the following command in the Package Manager Console

Install-Package AppForSharePointWebToolkit
