Official MCTS: SQL 2008 Business Intelligence Boot Camp - Benefits and Goals:

Our six (6) day intense Official MCTS: SQL 2008 Business Intelligence Boot Camp provides information technology professionals with the knowledge and skills necessary to design and deploy SQL Server 2008 database solutions to the enterprise. This is an accelerated immersion course, designed for computer professionals that require effective, real-world skill-building and timely certification.

Only Intense School’s Official MCTS: SQL 2008 Business Intelligence Boot Camp offers you the following benefits:
  •    Our definitive, TOTAL immersion boot camp experience

  •    In-person access to the top experts in the industry

  •    Highly accomplished status as a distinguished MCTS

Here’s what you’ll have access to:

Each student will be installing SQL Server 2008 Analysis Services, developing Analysis Services Solutions, debugging error handling in an Integration Services package, administering Reporting Services and learn how to implement custom Reporting Services applications.. Evening Sessions are not mandatory, but our world-class instructors are there to see the concepts of the day are reinforced through proper self paced exercises and practice materials.

Professional Exams and Certifications:

Intense School’s Official MCTS: SQL 2008 Business Intelligence Boot Camp extensively prepares students for the following Microsoft Certified Technology Specialist exam:

  •    Exam 70-448: TS: Microsoft SQL Server 2008 Business Intelligence – Implementation and Maintenance

You will achieve the following premium certification:

  •    MCTS: SQL Server 2008, Business Intelligence Development and Maintenance

    • Who Should Attend

    The audience for this course includes people with experience administering, supporting or designing relational databases and a working knowledge of Transact-SQL.  Preferably, students have already taken training towards and have passed Exam 70-431: TS: Microsoft SQL Server 2005 - Implementation and Maintenance.

Prerequisites?

In addition to having 2 years or more working with SQL 2000 or 2005, and preferably the MCTS: Microsoft SQL Server 2005 - Implementation and Maintenance certification, students must have:


  • Conceptual understanding of OLAP solutions.
  • Experience navigating the Microsoft Windows Server environment.
  • Experience with Windows services (starting and stopping).
  • Experience creating service accounts and permissions.
    • Experience with Microsoft SQL Server, including:
    • SQL Server Agent.
    • SQL Server accounts (users and permissions).
    • SQL Server query language (SELECT, UPDATE, INSERT, and DELETE).
    • SQL Server System tables.
