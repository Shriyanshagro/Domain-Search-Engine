        Das richtige IT-Training finden

        Egal, ob Sie erfahrener Experte oder neu auf diesem Gebiet sind: Microsoft IT-Trainings vermitteln Ihnen das nötige Know-how für ein bestimmtes Produkt oder eine Technologie. Wenn Sie von einem unserer Microsoft Certified Trainer geschult werden, können Sie sicher sein, dass das Erlernte korrekt, vollständig und auf dem neuesten Stand ist. Wählen Sie ein herkömmliches Präsenztraining oder eine der neuen Offizieller Microsoft On-Demand-Kurs (MOC On-Demand)-Optionen aus, bei denen Sie mit Ihrem eigenen Stundenplan, in einem selbstbestimmten Tempo und an Ihrem eigenen Platz lernen können.

        Neu in der IT? Erlernen Sie zunächst die Grundlagen der Technologie mithilfe der Online-Bildungs- und -Trainingsressourcen, die für die Zertifizierungen zum Microsoft Technology Associate (MTA) zur Verfügung stehen. Klicken Sie unten einfach auf eines der Trainings und finden Sie das gewünschte Training unter Optionen für die Vorbereitung.
