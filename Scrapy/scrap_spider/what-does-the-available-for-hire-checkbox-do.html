What does the 'Available for hire' checkbox do?

  • mac
  • windows
  • linux
  • all

Use the Available for hire checkbox to view GitHub Jobs posts within GitHub.

The GitHub Jobs board is a great way to find employment in tech. You can opt to see jobs posted there on your GitHub dashboard.

  1. In the top right corner of any page, click your profile photo, then click Settings.
  2. Under Jobs Profile, select Available for hire, then click Save jobs profile.
